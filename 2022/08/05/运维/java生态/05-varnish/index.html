Varnish是一款高性能的开源HTTP加速器，具有反向代理，缓存的功能。

1.部署Varmish缓存服务器

官网： http://varnish-cache.org/

测试，varnish命令正常

2.修改代理配置文件

安装完成后，没有配置文件，需要自己从安装包复制一份

3.启动varnish

4.varnish 缓存数据的存储地方

varnishd -s malloc,128M 默认内存缓存，定义varnish使用内存作为缓存 ,空间128M
varnishd -s file,/var/lib/varnish_strong.bin,1G 定义varnish使用文件作为缓存，空间1G

5.查看日志

//varnish日志，比较详细

//访问日志

6.更新缓存数据

在后台web服务器更新页面内容后，用户访问代理服务器看到的还是之前的数据， 说明缓存中的数据过期了需要更新 (默认也会自动更新，但非实时更新,3,5分钟)。
varnishadm
ban req.url ~ test.html //精确清空url
ban req.url ~ .* //清空缓存数据，支持正则表达式
