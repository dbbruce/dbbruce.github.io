GitеҲҶж”Ҝж“ҚдҪң

1.еёёи§Ғзҡ„еҲҶж”Ҝи§„иҢғеҰӮдёӢ

MASTERеҲҶж”Ҝ(MASTERжҳҜдё»еҲҶж”ҜпјҢжҳҜд»Јз Ғзҡ„ж ёеҝғ
DEVELOPеҲҶж”Ҝ (DEVELOPжңҖж–°ејҖеҸ‘жҲҗжһңзҡ„еҲҶж”Ҝ)еӨ§еҠҹиғҪ
RELEASEеҲҶж”Ҝ (дёәеҸ‘еёғж–°дә§е“Ғи®ҫзҪ®зҡ„еҲҶж”Ҝ)
HOTFIXеҲҶж”Ҝ (дёәдәҶдҝ®еӨҚиҪҜд»¶BUGзјәйҷ·зҡ„еҲҶж”Ҝ)
FEATUREеҲҶж”Ҝ (дёәејҖеҸ‘ж–°еҠҹиғҪи®ҫзҪ®зҡ„еҲҶж”Ҝ)е°ҸеҠҹиғҪ

2.жҹҘзңӢдҪҚдәҺйӮЈдёӘеҲҶж”Ҝ

  • жҹҘзңӢеҪ“еүҚеҲҶж”Ҝ
    git status жҹҘзңӢжүҖжңүеҲҶж”ҜдҝЎжҒҜ
  • еҲӣе»әеҲҶж”Ҝ
  • еҲҮжҚўеҲҶж”Ҝ
  • еҗҲе№¶еҲҶж”Ҝ
    е°Ҷhotfixдҝ®ж”№зҡ„ж•°жҚ®еҗҲе№¶еҲ°masterеҲҶж”Ҝ
    жіЁж„ҸпјҢеҗҲе№¶еүҚеҝ…йЎ»иҰҒе…ҲеҲҮжҚўеҲ°masterеҲҶж”ҜпјҢз„¶еҗҺеҶҚжү§иЎҢmergeе‘Ҫд»Ө
  • и§ЈеҶізүҲжң¬еҲҶж”Ҝзҡ„еҶІзӘҒй—®йўҳ жүӢеҠЁи§ЈеҶіеҶІзӘҒ еҶҚж¬ЎжҸҗдәӨпјҢи§ЈеҶіеҶІзӘҒ жӣҙж–°еҲҶж”Ҝ
