1.й»ҳи®Өзҡ„еҒҘеә·жЈҖжҹҘ

Kubernetesй»ҳи®Өзҡ„еҒҘеә·жЈҖжҹҘжңәеҲ¶пјҡ жҜҸдёӘе®№еҷЁеҗҜеҠЁж—¶йғҪдјҡжү§иЎҢдёҖдёӘиҝӣзЁӢпјҢ жӯӨиҝӣзЁӢз”ұDockerfileзҡ„CMDжҲ–ENTRYPOINTжҢҮе®ҡгҖӮ еҰӮжһңиҝӣзЁӢйҖҖеҮәж—¶иҝ”еӣһз Ғйқһйӣ¶пјҢ еҲҷи®Өдёәе®№еҷЁеҸ‘з”ҹж•…йҡңпјҢ Kubernetesе°ұдјҡж №жҚ®restartPolicyзӯ–з•ҘеҶіе®ҡжҳҜеҗҰйҮҚеҗҜе®№еҷЁгҖӮ

еңЁдёҠйқўзҡ„дҫӢеӯҗдёӯпјҢ е®№еҷЁиҝӣзЁӢиҝ”еӣһеҖјйқһйӣ¶пјҢ KubernetesеҲҷи®Өдёәе®№еҷЁеҸ‘з”ҹж•…йҡңпјҢ йңҖиҰҒйҮҚеҗҜгҖӮ жңүдёҚе°‘жғ…еҶөжҳҜеҸ‘з”ҹдәҶж•…йҡңпјҢ дҪҶиҝӣзЁӢе№¶дёҚдјҡйҖҖеҮәгҖӮ жҜ”еҰӮи®ҝй—®WebжңҚеҠЎеҷЁж—¶жҳҫзӨә500еҶ…йғЁй”ҷиҜҜпјҢ еҸҜиғҪжҳҜзі»з»ҹи¶…иҪҪпјҢ д№ҹеҸҜиғҪжҳҜиө„жәҗжӯ»й”ҒпјҢ жӯӨж—¶httpdиҝӣзЁӢе№¶жІЎжңүејӮеёёйҖҖеҮәпјҢ еңЁиҝҷз§Қжғ…еҶөдёӢйҮҚеҗҜе®№еҷЁеҸҜиғҪжҳҜжңҖзӣҙжҺҘгҖҒ жңҖжңүж•Ҳзҡ„и§ЈеҶіж–№жЎҲгҖӮ

k8s жҸҗдҫӣдәҶдёүз§ҚжқҘе®һзҺ°е®№еҷЁжҺўжөӢзҡ„ж–№жі•пјҢеҲҶеҲ«жҳҜпјҡ

  • 1гҖҒstartupProbeпјҡеҗҜеҠЁжҺўжөӢпјҢжҺўжөӢе®№еҷЁдёӯзҡ„еә”з”ЁжҳҜеҗҰе·Із»ҸеҗҜеҠЁгҖӮеҰӮжһңжҸҗдҫӣдәҶеҗҜеҠЁжҺўжөӢ(startup probe)пјҢеҲҷзҰҒз”ЁжүҖжңүе…¶д»–жҺўжөӢпјҢзӣҙеҲ°е®ғжҲҗеҠҹдёәжӯўгҖӮеҰӮжһңеҗҜеҠЁжҺўжөӢеӨұиҙҘпјҢkubelet е°ҶжқҖжӯ»е®№еҷЁпјҢе®№еҷЁжңҚд»Һе…¶йҮҚеҗҜзӯ–з•ҘиҝӣиЎҢйҮҚеҗҜгҖӮеҰӮжһңе®№еҷЁжІЎжңүжҸҗдҫӣеҗҜеҠЁжҺўжөӢпјҢеҲҷй»ҳи®ӨзҠ¶жҖҒдёәжҲҗеҠҹSuccessгҖӮ
  • 2гҖҒlivenessProbeпјҡеӯҳжҙ»жҺўжөӢпјҢз”ЁжҢҮе®ҡзҡ„ж–№ејҸпјҲexecгҖҒtcpгҖҒhttpпјүжЈҖжөӢpodдёӯзҡ„е®№еҷЁжҳҜеҗҰжӯЈеёёиҝҗиЎҢпјҢеҰӮжһңжЈҖжөӢеӨұиҙҘпјҢеҲҷи®Өдёәе®№еҷЁдёҚеҒҘеә·пјҢйӮЈд№ҲKubeletе°Ҷж №жҚ®Podдёӯи®ҫзҪ®зҡ„ restartPolicyзӯ–з•ҘжқҘеҲӨж–ӯPod жҳҜеҗҰиҰҒиҝӣиЎҢйҮҚеҗҜж“ҚдҪңпјҢеҰӮжһңе®№еҷЁй…ҚзҪ®дёӯжІЎжңүй…ҚзҪ® livenessProbeпјҢKubelet е°Ҷи®Өдёәеӯҳжҙ»жҺўй’ҲжҺўжөӢдёҖзӣҙдёәsuccessпјҲжҲҗеҠҹпјүзҠ¶жҖҒгҖӮ
  • 3гҖҒreadnessProbeпјҡе°ұз»ӘжҖ§жҺўй’ҲпјҢз”ЁдәҺжЈҖжөӢе®№еҷЁдёӯзҡ„еә”з”ЁжҳҜеҗҰеҸҜд»ҘжҺҘеҸ—иҜ·жұӮпјҢеҪ“жҺўжөӢжҲҗеҠҹеҗҺжүҚдҪҝPodеҜ№еӨ–жҸҗдҫӣзҪ‘з»ңи®ҝй—®пјҢе°Ҷе®№еҷЁж Үи®°дёәе°ұз»ӘзҠ¶жҖҒпјҢеҸҜд»ҘеҠ еҲ°podеүҚз«ҜиҙҹиҪҪпјҢеҰӮжһңжҺўжөӢеӨұиҙҘпјҢеҲҷе°Ҷе®№еҷЁж Үи®°дёәжңӘе°ұз»ӘзҠ¶жҖҒпјҢдјҡжҠҠpodд»ҺеүҚз«ҜиҙҹиҪҪз§»йҷӨгҖӮ

зңҹжӯЈзҡ„еҗҜеҠЁйЎәеәҸ

еҸҜд»ҘиҮӘе®ҡд№үеңЁpodеҗҜеҠЁж—¶жҳҜеҗҰжү§иЎҢиҝҷдәӣжЈҖжөӢпјҢеҰӮжһңдёҚи®ҫзҪ®пјҢеҲҷжЈҖжөӢз»“жһңеқҮй»ҳи®ӨдёәйҖҡиҝҮпјҢеҰӮжһңи®ҫзҪ®пјҢеҲҷйЎәеәҸдёәstartupProbe>readinessProbeе’ҢlivenessProbeпјҢreadinessProbeе’ҢlivenessProbeжҳҜе№¶еҸ‘е…ізі»

е®ҳж–№ж–ҮжЎЈпјҡCaution: Liveness probes do not wait for readiness probes to succeed. If you want to wait before executing a liveness probe you should use initialDelaySeconds or a startupProbe

  • д№ҹе°ұжҳҜ Liveness probes е№¶дёҚдјҡзӯүеҲ° Readiness probes жҲҗеҠҹд№ӢеҗҺжүҚиҝҗиЎҢпјҢж №жҚ®дёҠйқўзҡ„е®ҳж–№ж–ҮжЎЈпјҢLiveness е’Ң readiness еә”иҜҘжҳҜжҹҗз§Қе№¶еҸ‘зҡ„е…ізі»гҖӮ

зӣ®еүҚLivenessProbeе’ҢReadinessProbeгҖҒstartupprobeжҺўжөӢйғҪж”ҜжҢҒдёӢйқўдёүз§ҚжҺўй’Ҳпјҡ

  • 1гҖҒexecпјҡеңЁе®№еҷЁдёӯжү§иЎҢжҢҮе®ҡзҡ„е‘Ҫд»ӨпјҢеҰӮжһңжү§иЎҢжҲҗеҠҹпјҢйҖҖеҮәз Ғдёә 0 еҲҷжҺўжөӢжҲҗеҠҹгҖӮ
  • 2гҖҒTCPSocketпјҡйҖҡиҝҮе®№еҷЁзҡ„ IP ең°еқҖе’Ңз«ҜеҸЈеҸ·жү§иЎҢ TCP жЈҖ жҹҘпјҢеҰӮжһңиғҪеӨҹе»әз«Ӣ TCP иҝһжҺҘпјҢеҲҷиЎЁжҳҺе®№еҷЁеҒҘеә·гҖӮ
  • 3гҖҒHTTPGetпјҡйҖҡиҝҮе®№еҷЁзҡ„IPең°еқҖгҖҒз«ҜеҸЈеҸ·еҸҠи·Ҝеҫ„и°ғз”Ё HTTP Getж–№жі•пјҢеҰӮжһңе“Қеә”зҡ„зҠ¶жҖҒз ҒеӨ§дәҺзӯүдәҺ200дё”е°ҸдәҺ400пјҢеҲҷи®Өдёәе®№еҷЁеҒҘеә·

жҺўй’ҲжҺўжөӢз»“жһңжңүд»ҘдёӢеҖјпјҡ

  • 1гҖҒSuccessпјҡиЎЁзӨәйҖҡиҝҮжЈҖжөӢгҖӮ
  • 2гҖҒFailureпјҡиЎЁзӨәжңӘйҖҡиҝҮжЈҖжөӢгҖӮ
  • 3гҖҒUnknownпјҡиЎЁзӨәжЈҖжөӢжІЎжңүжӯЈеёёиҝӣиЎҢгҖӮ

PodжҺўй’Ҳзӣёе…ізҡ„еұһжҖ§пјҡ

жҺўй’Ҳ(Probe)жңүи®ёеӨҡеҸҜйҖүеӯ—ж®өпјҢеҸҜд»Ҙз”ЁжқҘжӣҙеҠ зІҫзЎ®зҡ„жҺ§еҲ¶Livenessе’ҢReadinessдёӨз§ҚжҺўй’Ҳзҡ„иЎҢдёә

  • initialDelaySecondsпјҡе®№еҷЁеҗҜеҠЁеҗҺиҰҒзӯүеҫ…еӨҡе°‘з§’еҗҺжҺўй’ҲејҖе§Ӣе·ҘдҪңпјҢеҚ•дҪҚвҖңз§’вҖқпјҢй»ҳи®ӨжҳҜ 0 з§’пјҢжңҖе°ҸеҖјжҳҜ 0
  • periodSecondsпјҡ жү§иЎҢжҺўжөӢзҡ„ж—¶й—ҙй—ҙйҡ”пјҲеҚ•дҪҚжҳҜз§’пјүпјҢй»ҳи®Өдёә 10sпјҢеҚ•дҪҚвҖңз§’вҖқпјҢжңҖе°ҸеҖјжҳҜ1
  • timeoutSecondsпјҡ жҺўй’Ҳжү§иЎҢжЈҖжөӢиҜ·жұӮеҗҺпјҢзӯүеҫ…е“Қеә”зҡ„и¶…ж—¶ж—¶й—ҙпјҢй»ҳи®Өдёә1пјҢеҚ•дҪҚвҖңз§’вҖқгҖӮ
  • successThresholdпјҡиҝһз»ӯжҺўжөӢеҮ ж¬ЎжҲҗеҠҹпјҢжүҚи®ӨдёәжҺўжөӢжҲҗеҠҹпјҢй»ҳи®Өдёә 1пјҢеңЁ Liveness жҺўй’Ҳдёӯеҝ…йЎ»дёә1пјҢжңҖе°ҸеҖјдёә1гҖӮ
  • failureThresholdпјҡ жҺўжөӢеӨұиҙҘзҡ„йҮҚиҜ•ж¬Ўж•°пјҢйҮҚиҜ•дёҖе®ҡж¬Ўж•°еҗҺе°Ҷи®ӨдёәеӨұиҙҘпјҢеңЁ readiness жҺўй’ҲдёӯпјҢPodдјҡиў«ж Үи®°дёәжңӘе°ұз»ӘпјҢй»ҳи®Өдёә 3пјҢжңҖе°ҸеҖјдёә 1

дёӨз§ҚжҺўй’ҲеҢәеҲ«пјҡ

  • ReadinessProbe е’Ң livenessProbe еҸҜд»ҘдҪҝз”ЁзӣёеҗҢжҺўжөӢж–№ејҸпјҢеҸӘжҳҜеҜ№ Pod зҡ„еӨ„зҪ®ж–№ејҸдёҚеҗҢпјҡ
  • readinessProbe еҪ“жЈҖжөӢеӨұиҙҘеҗҺпјҢе°Ҷ Pod зҡ„ IP:Port д»ҺеҜ№еә”зҡ„ EndPoint еҲ—иЎЁдёӯеҲ йҷӨгҖӮ
  • livenessProbe еҪ“жЈҖжөӢеӨұиҙҘеҗҺпјҢе°ҶжқҖжӯ»е®№еҷЁе№¶ж №жҚ® Pod зҡ„йҮҚеҗҜзӯ–з•ҘжқҘеҶіе®ҡдҪңеҮәеҜ№еә”зҡ„жҺӘж–ҪгҖӮ

  • /tmp/healthy ж–Үд»¶еҲ йҷӨеҗҺпјҢжЈҖжөӢеӨұиҙҘпјҢйҮҚеҗҜpod

  • дёҠйқў Pod дёӯеҗҜеҠЁзҡ„е®№еҷЁжҳҜдёҖдёӘ SpringBoot еә”з”ЁпјҢе…¶дёӯеј•з”ЁдәҶ Actuator з»„д»¶пјҢжҸҗдҫӣдәҶ /actuator/health еҒҘеә·жЈҖжҹҘең°еқҖпјҢеӯҳжҙ»жҺўй’ҲеҸҜд»ҘдҪҝз”Ё HTTPGet ж–№ејҸеҗ‘жңҚеҠЎеҸ‘иө·иҜ·жұӮпјҢиҜ·жұӮ 8081 з«ҜеҸЈзҡ„ /actuator/health и·Ҝеҫ„жқҘиҝӣиЎҢеӯҳжҙ»еҲӨж–ӯпјҡ
  • д»»дҪ•еӨ§дәҺжҲ–зӯүдәҺ200дё”е°ҸдәҺ400зҡ„д»Јз ҒиЎЁзӨәжҺўжөӢжҲҗеҠҹгҖӮ

  • scheme: з”ЁдәҺиҝһжҺҘhostзҡ„еҚҸи®®пјҢй»ҳи®ӨдёәHTTPгҖӮ
  • hostпјҡиҰҒиҝһжҺҘзҡ„дё»жңәеҗҚпјҢй»ҳи®ӨдёәPod IPпјҢеҸҜд»ҘеңЁhttp request headдёӯи®ҫзҪ®hostеӨҙйғЁгҖӮ
