如何重载预制数据

2023年08月26日
项目 > 医药公司ERP > 业务

涉及的知识点

  • 1、模型表: ir_model

    select * from ir_model where model =’med.pur.request.line1’;

  • 2、模型字段表: ir_model_fields

    select * from ir_model_fields where model =’med.pur.request.line1’ and name=’request_org_id’ ;

  • 3、预制数据表: ir_model_data；注意name字段，对应xml文件中的id内容

    select * from ir_model_data where name = ‘med_pur_request_line1_request_org’;

预制数据文件

  • 会在orgs_config表中写入5条数据
  • model - 数据表名
  • name - 表字段名

    …/data/xxx_data.xml

重载预制数据步骤

  • 1、noupdate=0，如果为1，xml中的record的更新并不会更新到数据库中
  • 2、删除 ir_model_data 表中涉及的数据

    DELETE from ir_model_data where name = ‘med_pur_request_line1_request_org’;

  • 3、删除涉及数据的表

select * from ir_model where model= ‘med.pur.request.line1’;

select id from orgs_config where model_id=’3583’;

delete from orgs_config where id in (1068,1067,1066,1065);
