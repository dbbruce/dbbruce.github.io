python特殊属性和方法

1. 对象属性相关
名称 描述
dict 对象的属性字典
slots 限制对象可拥有的属性
getattr(self, name) 获取不存在的属性时调用
setattr(self, name, value) 设置属性时调用
delattr(self, name) 删除属性时调用
getattribute(self, name) 获取任何属性时调用（优先
dir(self) 返回对象的属性列表
2. 描述符协议
名称 描述
get(self, instance, owner) 获取属性值
set(self, instance, value) 设置属性值
delete(self, instance) 删除属性
set_name(self, owner, name) 设置描述符名称
3. 类创建相关
名称 描述
init_subclass(cls) 类被继承时调用
class_getitem(cls, item) 支持泛型类型提示
mro_entries(self, bases) 自定义MRO条目
4. 实例化相关
名称 描述
new(cls[, …]) 创建实例（构造器）
init(self[, …]) 初始化实例
del(self) 析构函数
5. 字符串表示
名称 描述
str(self) str()和print()调用
repr(self) repr()和交互式环境显示
format(self, format_spec) format()函数调用
bytes(self) bytes()函数调用
6. 比较运算
名称 描述
eq(self, other) ==
ne(self, other) !=
lt(self, other) <
le(self, other) <=
gt(self, other) >
ge(self, other) >=
7. 算术运算
名称 描述
add(self, other) +
sub(self, other) -
mul(self, other) *
matmul(self, other) @ (Python 3.5+)
truediv(self, other) /
floordiv(self, other) //
mod(self, other) %
pow(self, other[, modulo]) **
8. 容器类型协议
名称 描述
len(self) len()
getitem(self, key) self[key]
setitem(self, key, value) self[key] = value
delitem(self, key) del self[key]
contains(self, item) in 操作符
iter(self) 返回迭代器
next(self) 迭代器下一个值
reversed(self) reversed()
9. 调用相关
名称 描述
call(self[, args…]) 让对象可调用
10. 上下文管理器
名称 描述
enter(self) with语句进入时调用
exit(self, exc_type, exc_val, exc_tb) with语句退出时调用
11. 序列化和拷贝
名称 描述
reduce(self) pickle序列化
reduce_ex(self, protocol) 支持协议的pickle
copy(self) 浅拷贝
deepcopy(self, memo) 深拷贝
12. 异步相关
名称 描述
await(self) 返回迭代器
aiter(self) 异步迭代器
anext(self) 异步下一个
aenter(self) 异步上下文管理器
aexit(self, …) 异步上下文管理器退出
13. 类相关属性
名称 描述
class 对象的类
bases 类的基类元组
name 类名
qualname 类的限定名
module 类所在的模块名
doc 类的文档字符串
