26.字符串处理

  • 截取
  • 替换
  1. 只替换第1个子串
    ${变量名/old/new}
  2. 替换全部子串
    ${变量名//od/new}
  • 删除
  1. 从左向右，最短匹配删除
    ${变量名#关键词} # 1个#号，删除从左侧第1个字符到最近的关键词“root”的部分，作通配符理
  2. 从左向右，最长匹配删除
    ${变量名##*关键词} # 2个#号， 删除从左侧第1个字符到最远的关键词“root”的部分 :
  3. 从右向左，最短匹配删除
    格式:${变量名%关键词*} # 一个% 删除从右侧最后1个字符到往左最近的关键词“:”的部分，*做通配符理解:
  4. 从右向左，最长匹配删除
    格式:${变量名%%关键词*} # 2个% 删除从右侧最后1个字符到往左最远的关键词“:”的部分
  5. 赋初值
    格式:${变量名:-初始值} # 当变量没有值时，使用初始化
