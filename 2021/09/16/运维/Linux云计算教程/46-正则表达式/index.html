46-正则表达式

2021年09月16日
运维 > linux云计算基础教程

1.基本正则列表 (grep)

正则符号 描述
^ 匹配行首
$ 匹配行尾
[] 集台，匹配集合中的任意单个字符
[^] 对集台取反
. 点 匹配任意单个字符
*米号 匹配前一个字符任意次数 [不允许单独使用]
{n,m} 匹配前一个字符n到m次
{n} 匹配前一个字符 n 次
{n, } 匹配前一个字符 n 次及以上
() 保留

2.扩展正则列表 (grep -E 或者 egrep)

正则符号 描述
+ 加号 最少匹配一次
？ 问号 最多匹配一次 或者0次
{n, m} 匹配n到m次
() 组合为整体， 保留 ,相当于复制，使用\1\2\3提取
| 竖号 或
\b 单词边界:空,空格,tab,特殊符号; 不能是字母，数字，下划线
