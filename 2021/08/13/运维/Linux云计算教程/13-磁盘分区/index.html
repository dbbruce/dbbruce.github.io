13-磁盘分区

2021年08月13日
运维 > linux云计算基础教程

1.显示磁盘信息

2.格式化磁盘

mkfs.ext4 /dev/sdb
mkfs.xfs /dev/sdb

3.挂载磁盘

mkdir /data1
mount /dev/sdb /data1
df 查看

4.开机挂载

注意：写错了，重启会导致系统打不开

配置文件：/etc/fstab
格式如下

设备路径 挂载点 文件类型 参数 备份标记 检测顺序
/dev/sdb /data1 ext4 default 0 0
/dev/mapper/centos-root / xfs defaults 0 0

检测文件写的是否正确：mount -a

5.添加磁盘

1、查看新磁盘
2、创建分区
3.刷新查看，分区

partprobe

4.格式化分区

mkfs.xfs /dev/sdb1

查看
blkid /dev/sdb1

5.开机自动挂载

vim /etc/fstab

6.测试挂载文件是否有问题，这个非常重要，如果有问题，会导致无法启动，一定要测试

mount -a
