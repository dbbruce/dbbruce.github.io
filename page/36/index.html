-PXC集群主要由两部分组成: Percona Server withXtraDB和Write Set Replication patches ( 同步、多主复制插件 )

数据强一致性、无同步延迟

没有主从切换操作，无需使用虚拟IP

分库分表

2017年02月10日
mysql

1.分库分表

  • 概念
    将存放在一台数据库服务器中的数据，按照持定方式进行拆分，分散存放到多台数据库服务器中，以达到了分散单台服务器负载的效果
  • 纵向切分
    将单个数据库的多个表按业务类型分类，分散存储到不同的数据库中
  • 横向切分
    按照表中指定字段的分片规则，将表记录按行切分分散存储到多个数据库中
  • 试验配置
  • 至少需要三个db，否则会报错
