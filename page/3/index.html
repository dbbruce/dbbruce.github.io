081-服务器及hcm应用相关操作

2025年07月18日
项目 > HCM教程
1、k8s命令
命令 功能
kubectl get nodes 查看所有节点
kubectl get pods 查看所有pod
kubectl get pods -o wide 查看所有pod的详细信息
kubectl get pods -n hcm 查看hcm命名空间下的所有pod
kubectl delete pod hcm-cloud-1231-34123-1231 重启pod，停止一个pod后会再重新启动一个pod，这个命令会等pod起来后才退出，执行后可以ctrl+c
kubectl exec -it hcm-cloud-1231-34123-1231 /bin/bash 进入pod
kubectl logs -f –tail=200 hcm-cloud-1231-34123-1231 查看pod的日志
kubectl get svc 查看所有svc,service名称
kubectl scale deploy hcm-cloud(svc的名称) –replicas=1 增加或减少deploy pod数量
