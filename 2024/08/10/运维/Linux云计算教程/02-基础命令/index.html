02-基础命令

2024年08月10日
运维 > linux云计算基础教程
留言

1.磁盘

/dev/sda sd代表磁盘类型，a代表第几块磁盘
/dev/hda
hd表示IDE设备，老设备，传输慢
sd表示scsi设备，新设备，传输快

2.命令行表示

[用户@主机名 工作目录]$

3.cat

查看系统版本
cat /etc/redhat-release
CentOS Linux release 7.9.2009 (Core)

4.CPU

lscpu

5.内存

[root@localhost ~]# cat /proc/meminfo

6.hostname

临时修改
[root@localhost ~]# hostname master
永久修改
[root@localhost ~]# vim /etc/hostname

7.ifconfig

网卡信息
enp0s3: flags=4163<UP,BROADCAST,RUNNING,MULTICAST> mtu 1500
inet 192.168.1.12 netmask 255.255.255.0 broadcast 192.168.1.255
inet6 2409:8a00:6037:3410:a579:ef16:4b1f:3b31 prefixlen 64 scopeid 0x0
inet6 fe80::f1a3:a1af:6006:3ae6 prefixlen 64 scopeid 0x20
ether 08:00:27:79:a3:b8 txqueuelen 1000 (Ethernet)
RX packets 1114 bytes 81591 (79.6 KiB)
RX errors 0 dropped 0 overruns 0 frame 0
TX packets 406 bytes 73419 (71.6 KiB)
TX errors 0 dropped 0 overruns 0 carrier 0 collisions 0

lo: 回环测试接口，ip用于为：127.0.0.1 ，代表自己，用于访问自己

8.touch

创建空文件
touch 1.txt

9.分屏阅读 less

less /etc/passwd
翻页 ：b 前/f 后
搜索 ： /root
搜索前后匹配： n 后/N 前
退出：q

10.head / tail

查看头5行
head -n 5 anaconda-ks.cfg 简写 head -5 anaconda-ks.cfg
尾部5行
tail -n 5 anaconda-ks.cfg 简写 tail -5 anaconda-ks.cfg
tail -f anaconda-ks.cfg

11.grep 输出包含指定字符串的内容

[root@localhost ~]# grep ‘root’ /etc/passwd
root:x:0:0:root:/root:/bin/bash
operator:x:11:0:operator:/root:/sbin/nologin

12.nmtui图形配置网络

13.vim

命名模式，插入模式，末行模式

命 — i — 插入模式 — ESC 退出
令
模
式 — : — 末行模式 — wq/!wq — ESC 退出

14.关机、重启

poweroff / reboot

15. “家目录”

~ 表示 “家目录”
/root 表示 管理员”家目录”
/home 表示 普通”家目录”

16. 添加用户

useradd tom1

17. ls

ls -h 显示单位
ls -a 显示隐藏文件
ls -l 长格式，显示全部信息

18.匹配符

*：任意个
？：一个

[a-z]:多个字符中的一个，若无忽略， 注意：不识别9以上

{a, min, xy}: 多组不同的字符串，全匹配

19.mkdir -p /test/test2/test3

创建父目录和子目录

20. -R 递归，显示所有子目录和子目录下的所有

ls -R /home

21. 删除 (危险命令)

rm
-r 递归删除
-f 强制删除
-i 提示是否删除
删除文件一定加i

22.grep 过滤

-i : 忽略大小写
-v : 取反
^root :以什么开头
root$ :以什么结尾

23.cp 复制

复制目录，-r 递归
cp -r /1 /2
