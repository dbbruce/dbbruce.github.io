076-时间管理业务总结

2025年07月12日
项目 > HCM教程

1.从【考勤设置】入手

设置考勤方案的基础项

业务开始于：考勤设置

自定义假期类型

考勤设置 → 自定义考勤项目 → 新增

加班类型

先点 → 初始化项目

2.【班次管理】

设置有哪些班，白班，晚班，三班倒，哺乳班等

3.【考勤方案】

设置考勤方案，排班时间等

4.【考勤档案】

人员与考勤方案关联

5.【考勤排班】

1、人员生成每天上什么班，生成排班后，可以手动修改
2、如果人员不能请假，查看下是不是没有排班

6.休假额度管理

注意：有一些请假单提示没有额度，就需要看下这里，对应人的额度计算，他是否有值。

1.导入人员的休假额度
2.【配置计算规则】
3.执行【额度计算】-标准产品是按钮，也可以自己加定时任务，比如夜里执行；
4.就可以请假了

7.休假申请

1.系统设置 - 基础服务 - 流程设置（旧版） - 更多操作 - 流程仓库 - 选择一个（比如：请休假） - 引入
注意：流程设置(旧版)里流程仓库有内容，另一个是空的

2.找到导入项，名字会默认带导入，启用发布

3.新增自定义应用


4.发布应用

8.考勤月报

1.需要计算，实时计算或则后台计算

9.各个流程

1、考勤业务主流程：工作日历 → 班次管理 → 考勤方案 → 考勤档案 → 考勤排班 → 考勤月报 → 工时台账
2、打卡流程：考勤机 → Beacon → 考勤点 → 移动打卡 → 补签申请 → 打卡台账
3、休假流程：休假项目 → 休假额度管理 → 休假申请 → 休假台账
4、加班流程：加班项目 → 加班类型 → 加班申请 → 加班台账

注意点：

1、请假如果提示没有假期，查看【休假额度管理】，里面有各类假期的额度

2、班次如果有多个时间段，比如上午和下午2个时间段，那么就需要打4次卡
