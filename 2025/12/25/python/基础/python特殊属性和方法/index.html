python特殊属性和方法

1. 对象属性相关
名称 描述
__dict__ 对象的属性字典,普通类的实例属性都存储在 dict 字典中
__slots__ 限制对象可拥有的属性
__getattr__(self, name) 当我们访问一个不存在的属性的时候，会抛出异常，提示我们不存在这个属性。而这个异常就是__getattr__方法抛出的，其原因在于他是访问一个不存在的属性的最后落脚点，作为异常抛出的地方提示出错再适合不过了。
__setattr__(self, name, value) 在对一个属性设置值的时候，会调用到这个函数，每个设置值的方式都会进入这个方法。
__delattr__(self, name) 删除属性时调用
__getattribute__(self, name) 获取任何属性时调用,__getattribute__是属性访问拦截器，就是当这个类的属性被实例访问时，会自动调用类的__getattribute__方法。当实例调用属性时，比如s1.name,会把name作为实参传进__getattribute__方法中，经过一系列操作后，再把name处理后的结果返回
__dir__(self) 返回对象的属性列表
2. 描述符协议
名称 描述
__get__(self, instance, owner) 获取属性值
__set__(self, instance, value) 设置属性值
__delete__(self, instance) 删除属性
__set_name__(self, owner, name) 设置描述符名称
3. 类创建相关
名称 描述
__init_subclass__(cls) 类被继承时调用
__class_getitem__(cls, item) 支持泛型类型提示
__mro_entries__(self, bases) 自定义MRO条目
4. 实例化相关
名称 描述
__new__(cls[, …]) 创建实例（构造器）
__init__(self[, …]) 初始化实例
__del__(self) 析构函数
5. 字符串表示
名称 描述
__str__(self) str()和print()调用
__repr__(self) repr()和交互式环境显示
__format__(self, format_spec) format()函数调用
__bytes__(self) bytes()函数调用
6. 比较运算
名称 描述
__eq__(self, other) ==
__ne__(self, other) !=
__lt__(self, other) <
__le__(self, other) <=
__gt__(self, other) >
__ge__(self, other) >=
7. 算术运算
名称 描述
__add__(self, other) +
__sub__(self, other) -
__mul__(self, other) *
__matmul__(self, other) @ (Python 3.5+)
__truediv__(self, other) /
__floordiv__(self, other) //
__mod__(self, other) %
__pow__(self, other[, modulo]) **
8. 容器类型协议
名称 描述
__len__(self) len()
__getitem__(self, key) self[key]
__setitem__(self, key, value) self[key] = value
__delitem__(self, key) del self[key]
__contains__(self, item) in 操作符
__iter__(self) 返回迭代器
__next__(self) 迭代器下一个值
__reversed__(self) reversed()
9. 调用相关
名称 描述
__call__(self[, args…]) 让对象可调用
10. 上下文管理器
名称 描述
__enter__(self) with语句进入时调用
__exit__(self, exc_type, exc_val, exc_tb) with语句退出时调用
11. 序列化和拷贝
名称 描述
__reduce__(self) pickle序列化
__reduce_ex__(self, protocol) 支持协议的pickle
__copy__(self) 浅拷贝
__deepcopy__(self, memo) 深拷贝
12. 异步相关
名称 描述
__await__(self) 返回迭代器
__aiter__(self) 异步迭代器
__anext__(self) 异步下一个
__aenter__(self) 异步上下文管理器
__aexit__(self, …) 异步上下文管理器退出
13. 类相关属性
名称 描述
__class__ 对象的类
__bases__ 类的基类元组
__name__ 类名
__qualname__ 类的限定名
__module__ 类所在的模块名
__doc__ 类的文档字符串
