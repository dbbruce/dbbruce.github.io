配置key 必填 说明
key 是 插件key值，唯一
type 是 插入点类型，格式：{pre/post操作}_{增删改操作} pre，post create_data,edit_data,remove_data,remove_datas
name 是 插件名称，标准产品插件写入插件路径，云函数填入云函数名称
plugin_type 否 插件类型，值为standard为标准代码 其他为云函数
meta_disabled 否 是否禁用，为true则禁用
