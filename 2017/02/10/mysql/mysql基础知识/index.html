1.常用的SQL命令分类

  • 管理数据库使用SQL( 结构化查询语言 )

4.记录（表）管理命令

  1. select * from 库名.表名 //查看表记录
  2. insert into 库名.表名 values(值1，值2),(值1，值2); insert into 库名.表名 (字段名1，字段名2) values(值1，值2),(值1，值2); //插入表记录,注意字段名一定不要加引号
  3. update 库名.表名 set 字段=值 where xxxx; //修改表记录
  4. delete from 表名; //删除表记录,清空
  5. drom table 表名; //删除表

7.日期时间类型

  • datetime - 时间范围大

范围：1000-01-01 00:00:00~9999-12-31 23:59:59
格式：yyyymmddhhmmss

  • timestamp - 时间范围小

范围：1970-01-01 00:00:00 ~ 2038-01-19 00:00:00
格式：yyyymmddhhmmss

  • 日期 date

范围:0001-01-01 ~ 9999-12-31
格式:yyyymmdd

  • 年 year

范围:1901~2155
格式 :yyyy

  • 时间 time

格式 : HHMMSS
